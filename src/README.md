# WATI Clone MVP

React + Vite single-file MVP for Jason's WhatsApp bot dashboard.

## How to run locally

1. `npm install`
2. `npm run dev`
3. Open http://localhost:5173

## Deploy
Recommended: Vercel (connect your GitHub repo and import).

Build command: `npm run build`
Output directory: `dist`

Notes:
- This frontend is an MVP with mock data. To enable real WhatsApp+OpenAI features, add a backend (server/webhook) and set environment variables there (e.g., OPENAI_API_KEY, GOOGLE_SHEET_ID).
